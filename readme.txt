График курсов валют доллара и евро по отношению к рублю.
Valute.sql  -- экспорт структуры и данных бд mysql, создайте пустую бд на хосте 
               и импортируйте этот файл
getdata.php -- скрипт для работы с базой(получение данных о валюте).Измените 
               в нем хост,бд,имя пользователя и пароль на Ваши.
class.php   -- класс php для получения данных о курсах валют из сервиса 
	       cbrates.rbc.ru.(должен находится в одной папке с getdata.php)
main.html   -- html страница с построением графика на стороне клиента,обращается
		к getdata.php. Также использует скрипты javascript в папке 
               scripts
